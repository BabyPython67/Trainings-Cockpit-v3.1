# Trainings-Cockpit v3.1

A comprehensive 16-week training dashboard for athletes preparing for volleyball tournaments and 5K running races. Built with React, running fully in the browser with local data persistence.

## Features

- **16-Week Training Plan** – Structured running, volleyball, and strength training schedule
- **Weekly Load Visualization** – Bar charts showing daily and weekly training intensity
- **Recovery Tracking** – Sleep, readiness scores, and nap tracking with readiness indicator
- **Nutrition Manager** – Kcal tracking with Mifflin-St Jeor BMR calculations
- **Pace & HR Analysis** – Running metrics with VO2max estimation and 5K Riegel projection
- **Theme Support** – Light and dark modes
- **Local Storage** – All data persists in your browser; no account needed
- **Export/Import** – Backup and restore your complete training data as JSON

## Quick Start

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. Start logging your training!

No installation, no build step, no server—just pure browser-based training management.

## Technologies

- **React 18.3.1** (UMD)
- **Recharts 2.12.7** (charts)
- **Tailwind CSS** (styling)
- **Lucide Icons** (icons)
- **Papa Parse** (CSV import)
- **Babel Standalone** (JSX transpilation)

## Data & Privacy

All data is stored locally in your browser's localStorage. Nothing is sent to external servers. You own your data.
